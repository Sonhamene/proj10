# 📚 Exercícios de Vetores em JavaScript

> “Sem usar funções de array” — o verdadeiro Dark Souls do JavaScript básico 💀  
> Tipo programar em 1997 enquanto alguém grita “NÃO USA `.filter()`!!!”

Este repositório contém exercícios de vetores desenvolvidos em JavaScript com foco em lógica de programação, estruturas de repetição e manipulação manual de arrays.

---

# 🧠 Conteúdo dos Exercícios

## ✅ Exercício 1 — Soma dos números pares

Programa que:

- Lê 10 números inteiros digitados pelo usuário
- Armazena os valores em um vetor
- Calcula a soma apenas dos números pares
- Exibe o resultado final

### 🎯 Conceitos utilizados
- Vetores
- `for`
- Condicionais (`if`)
- Operador módulo `%`

---

## 🎲 Exercício 2 — Vetor aleatório

Programa que:

- Preenche automaticamente um vetor com 7 números aleatórios
- Exibe todos os números gerados

### 🎯 Conceitos utilizados
- `Math.random()`
- Vetores
- Estruturas de repetição

---

## 🔍 Exercício 3 — Contagem de ocorrências

Programa que:

- Solicita 10 números inteiros ao usuário
- Armazena os valores em um vetor
- Pede outro número para busca
- Verifica quantas vezes esse número aparece no vetor

### 🎯 Conceitos utilizados
- Vetores
- Comparação de valores
- Contadores
- Loops

---

## 🚫♻️ Exercício 4 — Remoção de duplicados

- Recebe 10 números do usuário
- Armazena os valores em um vetor
- Cria um novo vetor removendo valores repetidos
- Exibe:
  - Vetor original
  - Vetor sem duplicatas

### 🎯 Conceitos utilizados
- Vetores
- Comparação manual de elementos
- Controle de repetição
- Lógica

> Sim, sem `.includes()`, sem `.filter()`, sem “vida fácil” 😭
